# mdisk
A bash script to partition and format a disk and mount the partition.  

## Getting Started
1. **Clone the repository**
    ```
    git clone https://github.com/mayankchak/mdisk.git
    cd mdisk/
    ```

2. **Make the script executable**
    ```
    chmod +x mdisk.sh
    ```
3. **Execute the script**  
    ```
    ./mdisk.sh
    ```
